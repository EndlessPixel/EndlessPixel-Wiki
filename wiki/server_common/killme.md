# 自杀

::: info
此文档中`<参数>`表示必须输入的参数，`[参数]`表示可选参数。
:::

💀 `/killme` 或 `/4` 或 `/killme:4`（自杀）

这三条指令是服务器为玩家提供的快捷自杀指令，功能完全一致，仅指令名称不同，满足不同玩家的操作习惯和使用偏好。输入任意一条指令后，系统会立即判定玩家角色死亡，无需额外确认步骤，角色会直接回到重生点或服务器设定的复活区域。该指令适用于多种游戏场景：比如角色不慎掉入无法逃脱的陷阱、被怪物围堵难以脱身，或是需要快速回到安全区域调整装备时，使用自杀指令能节省大量脱困时间。

::: warning
所有命令区分大小写，请正确输入。遇到问题可联系管理员帮助。
:::
