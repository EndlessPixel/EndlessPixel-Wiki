# 传送

::: info
文档中 `<参数>` 为必填参数，`[参数]` 为可选参数。所有指令**区分大小写**，输入时请确保准确。
:::

- `/rtp` — 随机传送到地图上的安全位置，适合探索。
- `/tpa <玩家名>` — 请求传送到指定玩家身边。
- `/tpahere <玩家名>` — 请求指定玩家传送到你身边。
- `/tpaignore <玩家名>` — 屏蔽某玩家的传送请求。
- `/tpauto` — 自动接受所有传送请求。
- `/tpaccept` — 接受当前收到的传送请求。
- `/tpatoggle [玩家名]` — 全局或针对某玩家开关传送请求权限。
- `/tpdeny` — 拒绝当前收到的传送请求。
- `/back` — 返回上一个位置。
- `/tpcancel <玩家名>` — 取消已发送给指定玩家的传送请求。

::: warning
所有命令区分大小写，请正确输入。遇到问题可联系管理员帮助。
:::
