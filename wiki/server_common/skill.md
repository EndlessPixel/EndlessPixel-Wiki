# 技能

::: info
文档中 `<参数>` 为必填参数，`[参数]` 为可选参数。所有指令**区分大小写**，输入时请确保准确。
:::

### ⭐ /skills（打开技能菜单，查看所有技能进度）
这是技能系统的核心查询指令，输入后会直接弹出可视化的技能菜单界面。菜单中会完整罗列服务器内所有开放的技能类型，比如耕种、采矿、战斗、锻造等，同时会清晰标注每一项技能的当前等级、已获取经验值和升级所需经验值，部分服务器还会显示技能的成长进度条。玩家通过这个菜单可以直观掌握自己全技能的发展情况，方便规划后续的技能升级方向，无论是新手玩家了解技能体系，还是老玩家统筹养成节奏，这条指令都是必备工具。

### ⭐ /stats（打开统计菜单，查看所有统计数据）
该指令用于调取角色的全维度统计面板，菜单内的数据覆盖范围远超技能进度，会详细记录角色的各项核心指标。其中包括技能相关的总等级、总经验，还有游戏行为类数据，比如击杀怪物数量、采集资源总量、完成任务次数、死亡次数等，部分服务器还会统计角色的成就解锁情况和特殊称号获取记录。这些统计数据不仅能让玩家回顾自己的游戏历程，还能作为判断角色发展方向的参考依据，是全面了解角色状态的重要指令。

### ⭐ /`[skill_name]`（直接打开特定技能的等级进度）
这是针对性的技能查询快捷指令，专为快速查看单一技能详情设计。玩家只需输入/加具体技能名，比如/farming（耕种）、/mining（采矿）、/combat（战斗），系统就会直接弹出该技能的专属信息面板。面板中会展示该技能的当前等级、经验进度、升级奖励，以及该技能对应的专属能力加成说明，无需在全技能菜单中翻找，大幅提升查询效率，尤其适合专注提升某一技能的玩家使用。

### ⭐ /sk lang `[lang]`（更改个人消息和菜单的语言）
这是技能系统的个性化语言设置指令，支持玩家根据自身需求切换界面和消息的显示语言。使用时需在指令后输入服务器支持的语言代码或语言名称，比如/sk lang zh-CN（中文）、/sk lang en-US（英文）。指令执行后，技能菜单、等级提升提示、经验获取通知等所有与技能系统相关的文本内容，都会切换为选定的语言。该指令满足了不同语言背景玩家的使用需求，让技能系统的操作体验更友好。

### ⭐ /abtoggle 或 /sk toggle（切换个人动作栏显示）
这组指令是技能系统的界面显示开关，用于控制技能相关的动作栏是否在游戏界面中显示。两条指令功能完全一致，玩家可根据习惯任选其一。当动作栏开启时，界面会显示常用技能的快捷释放按钮、技能冷却时间、当前可使用的技能特效等信息；关闭后则会隐藏这些内容，让游戏界面更简洁。该指令采用“一键切换”逻辑，每输入一次就会在“开启”和“关闭”状态间切换，适配不同玩家的界面整洁偏好。

### ⭐ /sk top `[page_num]` 或 /skilltop `[page_num]`（查看所有技能等级总和排行榜）
这组指令用于查看服务器的全技能总等级排行榜，完整指令/skilltop和简写指令/sk top功能相同。排行榜会按照所有玩家的技能等级总和进行降序排列，展示玩家昵称、总等级数值、排名位次等信息。指令后的[页码]参数用于翻页，当排行榜人数较多时，输入/sk top 2即可查看第二页的排名内容。通过这条指令，玩家可以直观了解自己在服务器中的技能养成整体水平，同时也能参考高排名玩家的技能发展策略。

### ⭐ /sk top `[skill_name]` `[page_num]`（查看特定技能排行榜）
这是单一技能的专项排名查询指令，能精准定位某一技能的服务器玩家排名情况。使用时需输入指令+具体技能名+页码参数，比如/sk top farming 1，系统会立即展示该技能的等级排行榜，按照技能等级从高到低排序，清晰列出上榜玩家的昵称、技能等级和排名。该指令适合专注于某一技能养成的玩家，既能查看自己在该技能领域的实力位置，也能学习顶尖玩家的升级技巧，是针对性提升技能的实用参考工具。

### ⭐ /sk top average `[page_num]`（查看平均技能等级排行榜）
这条指令用于调取服务器的技能平均等级排行榜，核心排名依据是玩家所有已解锁技能的等级平均值。输入指令后，系统会自动计算每位玩家的技能等级总和，再除以技能数量得出平均值，随后按照平均值降序排列生成排行榜。搭配`[page_num]`参数可以翻页查看更多排名，比如/sk top average 3就是查看第三页的平均等级排名。该排行榜能反映玩家的技能养成均衡度，避免出现“偏科”严重的情况，为追求全面发展的玩家提供参考。

### ⭐ /sk rank 或 /skillrank（查看自己的技能排名）
这组指令是专属的个人排名查询工具，两条指令功能完全一致，无需额外参数即可使用。输入指令后，系统会快速返回玩家的两项核心排名数据：一是全技能总等级的服务器排名，二是当前各项技能的单项排名概况。部分服务器还会在结果中标注玩家与上一名次的等级差距，方便玩家制定冲榜目标。该指令操作简便，无需翻阅冗长的排行榜，就能快速掌握自己的技能排名定位。

### ⭐ /mana（显示当前法力值）
这是法力值的快捷查询指令，输入后会在游戏聊天框中直接显示角色当前的法力值总量和剩余数值。法力值是释放技能的核心消耗资源，无论是战斗技能、采集辅助技能还是锻造附魔技能，都需要消耗一定的法力值。通过这条指令，玩家可以随时掌握自己的法力储备情况，避免在关键时刻因法力不足无法释放技能。部分服务器还会在显示结果中附带法力值恢复速度的提示，帮助玩家更好地规划技能释放节奏。

### ⭐ /sk claimitems（打开物品认领菜单）
这是技能系统的福利领取指令，专门用于领取技能升级带来的奖励物品。当玩家的某项技能提升到指定等级时，会解锁对应的等级奖励，可能是稀缺资源、技能专属道具、经验加成药剂等。输入该指令后会弹出物品认领菜单，菜单中会列出所有玩家已解锁但未领取的技能奖励，玩家只需点击对应的奖励选项，即可将物品收入囊中。该指令解决了奖励遗漏的问题，确保玩家的每一次技能升级都能收获对应的福利。

### ⭐ /sk sources `[skill_name]` `[sort]`（查看技能经验获取方式）
这条指令是技能升级的攻略查询工具，用于查看指定技能的所有经验获取途径。使用时需输入指令+目标技能名，比如/sk sources farming，系统会列出该技能的经验获取方式，例如耕种特定作物、收获成熟植物、完成耕种相关任务等。指令后的`[sort]`参数支持按经验收益高低排序，输入/sk sources mining sort即可查看采矿技能中收益最高的经验获取途径。该指令能帮助玩家避开低效的升级方式，快速找到最适合自己的技能养成路径。

### ⭐ /sk help（显示技能命令帮助）
这是技能系统的指令指南查询入口，输入后会在聊天框中展示所有技能相关指令的完整信息。内容包括每条指令的准确格式、参数要求、功能说明以及权限限制，比如标注哪些指令是管理员专属、哪些指令对所有玩家开放。无论玩家是忘记了某条指令的使用方法，还是想了解技能系统的全部功能，都可以通过这条指令获取清晰的指引，是新手玩家快速上手技能系统的必备指令。

::: warning
所有命令区分大小写，请正确输入。遇到问题可联系管理员帮助。
:::
