# 技能

::: info
文档中 `<参数>` 为必填参数，`[参数]` 为可选参数。所有指令**区分大小写**，输入时请确保准确。
:::

## ⭐ 技能指令

- `/skills` — 打开技能菜单，查看所有技能等级与经验进度。
- `/stats` — 打开统计菜单，查看全维度游戏数据（击杀、采集、死亡等）。
- `/`**`[skill_name]`** — 直接查看特定技能详情，如 `/farming`、`/mining`。
- `/sk lang [lang]` — 更改技能系统界面语言（如 `zh-CN`、`en-US`）。
- `/abtoggle` 或 `/sk toggle` — 切换技能动作栏显示（冷却、快捷释放等）。
- `/sk top [page_num]` 或 `/skilltop [page_num]` — 查看技能总等级排行榜。
- `/sk top <skill_name> [page_num]` — 查看指定技能的等级排行榜。
- `/sk top average [page_num]` — 查看平均技能等级排行榜（反映技能均衡度）。
- `/sk rank` 或 `/skillrank` — 查看自己的总等级排名及单项概况。
- `/mana` — 显示当前法力值及恢复信息。
- `/sk claimitems` — 打开物品认领菜单，领取技能升级奖励。
- `/sk sources <skill_name> [sort]` — 查看技能经验获取方式，支持按收益排序。
- `/sk help` — 显示所有技能命令的帮助信息。

::: warning
所有命令区分大小写，请正确输入。遇到问题可联系管理员帮助。
:::
