::: info

## 👤 皮肤指令

- `/skin set <正版玩家名>` — 将自己的皮肤设置为指定正版玩家的皮肤。
- `/skin url <链接>` — 通过图片链接设置自己的皮肤（需服务器支持格式）。
- `/skins` — 打开皮肤菜单，浏览并选择皮肤。
- `/skin clear` — 清除当前皮肤，恢复为默认皮肤。
- `/skin update` — 更新皮肤资源，修复显示异常。
- `/skin search <关键词>` — 按关键词搜索皮肤库中的皮肤。
- `/skin <皮肤名>` — 直接更换为指定名称的皮肤（需名称完全匹配）。
- `/skin help` — 显示所有皮肤命令的帮助信息。

::: warning
所有命令区分大小写，请正确输入。遇到问题可联系管理员帮助。
:::
