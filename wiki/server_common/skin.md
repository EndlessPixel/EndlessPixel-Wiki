# 皮肤

::: info
文档中 `<参数>` 为必填参数，`[参数]` 为可选参数。所有指令**区分大小写**，输入时请确保准确。
:::

### 👤 /skin set <正版玩家名>（设置皮肤）
这组指令是服务器皮肤管理的核心功能，分为两种使用场景，满足单人皮肤设置和他人皮肤配置的需求。无目标玩家参数的/skin set <正版玩家名>，是为自身设置皮肤的指令，输入的正版玩家名需对应正版账号的公开皮肤，系统会自动同步该账号的皮肤资源并应用到自身角色上。

### 👤 /skin url <链接>（通过链接设置自己的皮肤）
该指令是为用户提供的自定义皮肤设置渠道，支持通过外部皮肤链接为自身角色配置专属皮肤。使用时需在指令后粘贴有效的皮肤文件链接，链接对应的文件需为服务器支持的格式，否则会提示“文件格式不兼容”。系统会自动解析链接并下载皮肤资源，校验通过后直接应用到用户的角色上。需要注意的是，该指令仅对自身角色生效，且部分服务器会对自定义皮肤进行合规性审核，违规内容的皮肤会被自动拦截，无法正常使用。

### 👤 /skins（打开皮肤菜单）
这是服务器皮肤管理的可视化操作入口，输入指令后会直接弹出皮肤菜单界面，无需记忆复杂的指令参数。菜单内通常分类展示服务器的所有皮肤资源，包括官方默认皮肤、活动限定皮肤、自定义上传皮肤等，用户可通过点击操作浏览、预览、选择皮肤，选中后一键应用到自身角色。该指令对所有玩家开放，操作简单直观，尤其适合不熟悉指令参数的新手玩家，是最便捷的皮肤更换方式。

### 👤 /skin clear（清除皮肤）
这组指令用于将角色皮肤恢复至初始状态，分为个人操作和权限操作两种模式。无参数的/skin clear是面向所有玩家的指令，输入后会清除自身当前使用的皮肤，角色会自动恢复为服务器默认的基础皮肤款式。

### 👤 /skin update（更新皮肤）
这组指令用于同步皮肤的最新版本，解决皮肤显示异常、版本过时等问题。无参数的/skin update是为自身皮肤更新的指令，当用户发现自己的角色皮肤出现贴图错误、加载不全等情况时，输入该指令即可触发皮肤资源的重新下载和同步，修复显示问题。

### 👤 /skin search <关键词>（搜索皮肤）
这是服务器皮肤资源的检索指令，专为快速查找特定皮肤设计。使用时在指令后输入皮肤相关的关键词，比如皮肤名称、风格标签、所属系列等，系统会自动在皮肤库中检索匹配的资源，并列出所有符合条件的皮肤名称及预览信息。用户可根据检索结果，搭配其他皮肤指令进行设置或更换。该指令支持模糊检索，即使输入的关键词不完整，也能匹配到相关皮肤，大幅提升查找效率。

### 👤 /skin <皮肤名>（更换自己的皮肤）
这是一款极简的皮肤更换指令，面向所有已熟悉服务器皮肤名称的玩家。使用时无需复杂参数，只需在指令后输入服务器内已有的皮肤名称，系统会自动校验皮肤名称的正确性，确认无误后直接将该皮肤应用到用户自身的角色上。该指令操作门槛低、执行速度快，适合频繁更换皮肤的玩家，需要注意的是，输入的皮肤名需与服务器内的命名完全一致，否则会提示“未找到该皮肤”。

### 👤 /skin help（显示皮肤命令帮助）
这是皮肤管理指令的查询入口，输入后系统会列出所有皮肤相关指令的完整信息，包括指令格式、参数要求、功能说明、权限限制等内容。无论玩家是新手还是老用户，当忘记某条指令的使用方法、不清楚参数如何填写时，都可以输入该指令获取详细指引。帮助信息会以清晰的列表形式展示，方便玩家快速查阅，是掌握皮肤管理所有功能的实用工具指令。

::: warning
所有命令区分大小写，请正确输入。遇到问题可联系管理员帮助。
:::
