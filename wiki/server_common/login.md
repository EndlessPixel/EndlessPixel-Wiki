# 登录与注册

::: info
文档中 `<参数>` 为必填参数，`[参数]` 为可选参数。所有指令**区分大小写**，输入时请确保准确。
:::

## 🔐 注册

- `/register <密码> <重复密码>` 或 `/reg <密码> <重复密码>` — 注册新账号。需包含数字、字母、特殊字符，长度达标。

## 🔐 登录

- `/login <密码>` 或 `/l <密码>` — 登录已注册账号。多次失败可能触发临时锁定。

## 🔐 修改密码

- `/changepassword <旧密码> <新密码> <重复新密码>` 或 `/changepw <旧密码> <新密码> <重复新密码>` — 修改账号密码。

## 🔐 绑定邮箱

- `/bindemail set <邮箱地址>` 或 `/bdmail set <邮箱地址>` — 绑定安全邮箱（需验证邮件，一个邮箱通常只能绑定一个账号）。
- `/bindemail verify <验证码>` 或 `/bdmail verify <验证码>` — 验证邮箱验证码，完成绑定。

## 🔐 忘记密码

- `/resetpassword forget` 或 `/repw forget` — 触发密码重置流程（需已绑定邮箱）。
- `/bindemail re <验证码> <新密码>` 或 `/bdmail re <验证码> <新密码>` — 使用邮箱验证码重置密码。

::: warning
所有命令区分大小写，请正确输入。遇到问题可联系管理员帮助。
:::
