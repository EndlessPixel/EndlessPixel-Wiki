# 附魔金苹果配方恢复

附魔金苹果是Minecraft中极为稀有的强力食物，能提供多种高额增益效果。

原版游戏中，该物品的合成配方（8 个金块 + 1 个苹果）已被移除

我们通过数据包恢复了经典合成配方，并**强化**了其效果，让它成为值得投入的终局道具。
