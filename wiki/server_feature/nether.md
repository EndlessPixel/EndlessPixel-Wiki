# 下界维度升级

## 🌋 生物群落深度优化

### 下界荒原重构
下界荒原迎来颠覆性生态改版，彻底摆脱以往单调的 “下界沙漠” 面貌，升级为充满危险与未知的洞穴式敌对生态。区域内新增大量壮观的下界尖刺结构，林立的石笋与钟乳石群纵横交错，配合厚重的黑石岩层与涌动的岩浆 “湖泊”，构建出危机四伏的探索环境。踏入这片区域的玩家，不仅要面对更密集的下界敌对生物，还要留意地形带来的坠落风险，每一步探索都充满挑战性。

### 森林生态增强
猩红森林与扭曲森林两大特色群落完成细节升级，新增散落的黑石岩石群、风化的倒下真菌茎残骸，以及新型蔓延生长的巨大真菌物种，让原本单一的菌菇地貌更具层次感。同时，服务器适度提升了群落内的植被密度，优化了真菌的分布规律，让整个生态系统更加丰富饱满，既提升了生物群落的真实感，也增加了隐藏资源点与探索路线，大幅提高了两大森林的探索价值。

## 🏰 史诗级新结构

### 猩红塔与扭曲塔
作为下界天际线的标志性宏伟建筑，猩红塔与扭曲塔的高度近100 个方块，矗立在对应的森林群落中，视觉冲击力极强。
- 丰厚探索奖励：塔内每层都设有专属战利品箱，宝箱中包含下界稀有资源、附魔装备及独特的功能性道具，塔顶宝箱更有概率开出服务器限定奖励；同时塔内设置特殊生物生成点，击败后可获得专属材料。
- 实用垂直交通：塔内设计了巧妙的垂直交通系统，通过螺旋阶梯与悬挂式平台连接各层，玩家可借助塔楼实现下界不同高度层之间的快速移动，堪称天然的下界 “电梯”。
- 环境完美融合：两座塔楼的建筑风格与对应森林群落高度适配，猩红塔的红色基调与猩红森林的菌菇色彩相得益彰，扭曲塔的诡异造型则与扭曲森林的扭曲地貌浑然一体，兼具视觉观赏性与探索实用性。
