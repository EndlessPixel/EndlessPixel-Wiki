# 末地维度升级

## 🌌 Nullscape末地维度升级
我们对末地维度进行了颠覆性的重构优化，核心亮点在于将末地高度大幅提升至384格，深度利用1.18版本的现代地形生成功能，打破了传统末地单调的浮空岛格局，打造出极具科幻感的多样化外星景观。

更新后的末地不再局限于零散的小型岛屿，取而代之的是规模宏大的破碎群岛、纵深交错的漂浮山谷、尖锐陡峭的结晶山峰等独特地形。每一片区域都拥有专属的地貌属性与视觉风格：破碎岛屿地带遍布悬空的岩块与裂隙，行走时需时刻提防坠落风险；漂浮山谷被透明的结晶植被覆盖，光线折射下宛如梦幻秘境；结晶山峰则由璀璨的紫晶与黑曜石构成，山顶还会刷新稀有资源与特殊生物。

不同区域的生态与危险程度也截然不同，部分区域会伴随高强度的末地风暴，还有的区域会生成专属的敌对变异生物，让每一次探索都充满新鲜感与挑战性，彻底颠覆玩家对末地的固有认知。

## 🏴 Explorify船难·末地新结构
新增的船难主题结构是末地维度的全新探索地标，这些因意外坠毁在末地的星际残骸，散落在破碎岛屿与漂浮山谷的边缘地带，与末地的外星地貌形成强烈的视觉反差。

船难结构的外观充满废弃科幻风格，锈蚀的金属外壳、破损的船舱、外露的机械部件，完美还原了星际载具坠毁后的破败感。结构内部设计了多层探索空间，不仅藏有大量末地稀缺的补给物资与高科技道具，还留存着记录船难原因的神秘日志；同时船舱的狭窄通道与隐藏隔间，会成为末影螨、潜影贝等生物的巢穴，玩家在探索时需兼顾资源收集与战斗避险。

这些船难结构不仅是末地探索的重要战利品来源，更以独特的叙事感，为末地维度增添了一层神秘的背景故事。
