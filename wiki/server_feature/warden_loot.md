# 更多循声守卫战利品
::: warning
以下内容不一定正确实装，我们仅确认安装了这些数据包，但实际上内容有没有生效，我们并不知道，因为暂无玩家进行测试

这么可怕的 Boss 难道不应该掉落比单个幽匿催化剂更刺激的东西吗？
这个数据包重新强化了循声守卫的掉落体系，让它成为一个真正值得挑战的终阶 Boss。

## 额外掉落
- 幽匿尖啸者（10% 概率掉落，带附魔光效与专属颜色名称，可召唤循声守卫）
- 回响碎片（低概率，**仅被复原指南针直接命中时才会掉落**）
- 音乐碟 5 碎片（极低概率直接掉落完整唱片）
- 多种随机矿石：铁、金、钻石、下界合金
- 概率掉落：循声守卫纹饰模板 / 静默盔甲纹饰模板
- 迅捷潜行 I–III 附魔书，或等价 30–50 级品质的随机附魔书
- 金苹果与附魔金苹果

## 新增进度（位于「冒险」标签页）
- 寂静深渊：进入深暗之域
- 潜行达人 100：原版进度
- 三分之一死亡：激活一个幽匿尖啸者
- 觉醒：惊扰黑暗中的巨型生物
- 重归寂静：击败循声守卫，暂时不会触发幽匿尖啸者
- 古城踪迹：进入远古城市
- 上古遗物：从远古城市盗取宝藏
- 迅捷潜行：装备带有迅捷潜行的护腿并潜行
- 召唤灵体：获得复原指南针
- 空城回响：在远古城市中播放音乐碟 5
