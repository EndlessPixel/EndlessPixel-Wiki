# BlazeandCave 成就包
## 🏆 超千项新进度，全面重塑生存挑战维度
BlazeandCave 进阶包为 Minecraft 服务器带来了一场前所未有的进度革命——新增超过 1000 项精心设计的进度挑战，将全服进度总数一举提升至 1202 个！这一规模不仅远超原版游戏的 98 项成就，更彻底打破了“生存即挖矿打怪”的传统循环。
无论是刚踏入方块世界的萌新，还是身经百战的硬核玩家，都能在这一庞大的挑战体系中找到属于自己的目标：
- 休闲玩家可循序渐进地完成趣味任务，享受探索与成长的乐趣；
- 竞技玩家则能在高难度挑战中一较高下，用实绩证明自己的操作、策略与耐心；
- 内容创作者也能从中挖掘大量剧情线索与玩法创意，打造独具特色的生存系列视频。
这套进度系统不再是简单的“打卡清单”，而是一张引导玩家深入体验 Minecraft 全维度玩法的动态地图，极大提升了服务器的长期可玩性、社区活跃度与内容竞争力。

## 📦 扩展附属：Enhanced Discoveries 强化探索包
本服务器额外搭载了 **BlazeandCave's Advancement Pack - Enhanced Discoveries** 非官方扩展包，在原有 1202 项进度基础上，**再新增超过 350 项全新成就**，进一步丰富挑战内容与探索目标。

扩展包特色：
- 新增 350+ 独立进度，包含普通任务、目标挑战、高阶挑战与隐藏成就
- 支持极限模式（Hardcore），可搭配专属硬核扩展使用
- 优化任务设计，鼓励玩家建造农场、搭建红石机械、制定战术、完成非常规操作
- 支持多语言，包含简体中文汉化，体验更友好
- 性能友好，在多人服务器环境下依然流畅运行

进度构成：
- 普通任务：31 项
- 目标挑战：102 项
- 高阶挑战：130 项
- 超级挑战：47 项
- 隐藏成就：46 项

该扩展与原版 BACAP 完全兼容，所有新增成就会直接整合进原有的成就体系，玩家无需额外设置，进入游戏即可体验更完整、更具深度的成就路线。

## 📋 多元创意融合，打造有温度、有记忆的进度体系
本进阶包的进度设计绝非机械堆砌，而是融合怀旧情怀、跨平台特色与原创巧思的结晶：
- 致敬经典：部分进度复刻自 1.12 以前版本的经典成就（如“开始了？”、“狙击手的对决”等），唤醒老玩家尘封的游戏记忆，让情怀与挑战并存；
- 打破平台壁垒：特别引入了基岩版（Bedrock）与主机版（Console Edition）独有的成就机制，例如“怪物猎人”、“需要来点薄荷糖？”等，让多平台玩家在 Java 服务器上也能重温熟悉挑战；
- 原创精神闪耀：更多进度源自主创团队 BlazeandCave 与多位社区贡献者的奇思妙想——从“用音符盒演奏完整旋律”到“在雷暴中徒手击杀闪电苦力怕”，每一项都兼具趣味性、叙事感与技术挑战，鼓励玩家跳出常规思维，尝试前所未有的玩法组合。
为方便玩家规划挑战路径，项目组提供了完整的进度查询工具与分类导航，支持按标签、难度、解锁条件等多维度筛选，助你高效制定“成就攻略”。
此外，进阶包对进度体系进行了结构性升级：在原版 5 个基础标签之上，全新构建了 16 个精细化分类，不仅为原有类别注入海量扩展内容，更针对红石、药水、生物群落等垂直领域开辟专属赛道，使整个进度树逻辑清晰、层次分明、易于探索。

## 🎯 16 大分类全覆盖，精准匹配每一种玩法偏好
无论你是建筑师、农夫、战士、探险家还是红石工程师，这里都有为你量身定制的挑战舞台：
1. BlazeandCave 的进步
游戏的“引路人”——涵盖从首次合成工作台到击败末影龙的核心里程碑，是所有玩家开启冒险的起点。
2. 采矿
深入地心，挑战极限！从煤炭到远古残骸，从精准采集到熔炉效率，考验你的资源勘探与冶炼技艺。
3. 建筑
从简陋木屋到空中城堡，从像素画到自动农场，用方块讲述你的创意故事。进度覆盖材料多样性、结构复杂度与美学完成度。
4. 农业
春种秋收，生生不息。不仅包含小麦、胡萝卜等基础作物，还涉及海泡菜、甜浆果、菌类等稀有植物的培育与自动化收割挑战。
5. 动物
驯服、繁殖、放牧、垂钓——畜牧达人与钓鱼高手的专属领域。甚至包括“用命名牌给所有可命名生物起名”这类细节控挑战！
6. 怪物
直面恐惧，征服黑暗！针对每一种敌对生物（包括灾厄村民、守卫者、猪灵等）设计独特击杀条件，如“无伤击败凋灵”、“在1秒内击杀5只幻翼”等高阶操作。
7. 武器
不只是剑与弓！进度涵盖三叉戟投掷、雪球击退、盾牌格挡反击，甚至“用TNT炸死自己以外的所有生物”等非常规战术，重新定义战斗艺术。
8. 生物群落
走遍主世界每一个角落：在繁茂洞穴种花、于恶地高原建塔、潜入深暗之域寻找幽匿……用脚步丈量世界的多样性。
9. 奇遇
拥抱随机性！村庄交易达成特定金额、触发袭击事件并获胜、找到林地府邸/海底神殿等结构，每一次探索都充满未知惊喜。
10. 红石
红石工程师的终极试炼场：从基础电路到全自动刷怪塔，从音乐播放器到逻辑计算机，进度层层递进，见证你从“电线工”成长为“方块程序员”。
11. 统计
数据控的天堂！累计行走1000公里、放置10万个方块、使用1万次工具……用数字记录你的 Minecraft 生涯。
12. 下面（下界）
烈焰与硫磺之地的专属挑战：狩猎烈焰人、穿越玄武岩三角洲、在堡垒遗迹中全身而退……下界不再是“过路站”，而是高风险高回报的冒险区。
13. 药水
炼金术大师之路：酿造所有基础药水、叠加多种效果、用喷溅药水击败特定怪物……让一瓶小小的药水成为扭转战局的关键。
14. 结束（末地）
终焉之地的终极篇章：击败末影龙只是开始，后续还有末地城掠夺、末地船探索、潜影贝狩猎、末地折跃门穿梭等深度内容，真正“打完”末地需要数十小时投入。
15. 超级挑战
仅限顶尖玩家！ 此分类中的进度需在击败末影龙后方可解锁，包含“无装备通关”、“单人速通全进度”、“用最弱武器击败最强Boss”等极限挑战，是服务器公认的“传奇成就”。
